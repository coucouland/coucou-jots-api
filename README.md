# Jot service
A service facilitating creation of jots in space and time

# Metadata
The following attributes help to define a jot

## Content
* Summary
* Description

## Time
* Start/end date
* Recurrence rules
* Date created
* Last modified

## Location (global)
* Lat/long
* Altitude
* Direction

## Location (relative to parent)
* deltaX, deltaY, deltaZ
